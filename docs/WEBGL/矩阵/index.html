视图矩阵​

通过目标点，视点，上方向，三个点计算视图矩阵。x,y 方向不变，z 方向相反。

  • 目标点:target→=(tx,ty,tz,tw)\overrightarrow{target} = (tx,ty,tz,tw)target​=(tx,ty,tz,tw)
  • 视点:eye→=(ex,ey,ez,ew)\overrightarrow{eye} = (ex,ey,ez,ew)eye​=(ex,ey,ez,ew)
  • 上方向:up→=(ux,uy,uz,uw)\overrightarrow{up} = (ux,uy,uz,uw)up​=(ux,uy,uz,uw)
  1. 以目标点为起点，视点为终点计算d→=eye→−target→=(dx,dy,dz,dw)\overrightarrow{d} = \overrightarrow{eye} - \overrightarrow{target} = (dx,dy,dz,dw)d=eye​−target​=(dx,dy,dz,dw)
  2. 将d→\overrightarrow{d}d归一化d→∗=1/∣d→∣\overrightarrow{d} *= 1 / |\overrightarrow{d}|d∗=1/∣d∣
  3. 利用d→\overrightarrow{d}d与上方向叉乘求出a→=d→×up→=(ax,ay,az,aw)\overrightarrow{a} = \overrightarrow{d} \times \overrightarrow{up} = (ax,ay,az,aw) a=d×up​=(ax,ay,az,aw)
  4. 将a→\overrightarrow{a}a归一化a→∗=1/∣a→∣\overrightarrow{a} *= 1 / |\overrightarrow{a}|a∗=1/∣a∣
  5. 利用d→\overrightarrow{d}d与上方向叉乘求出b→=d→×a→=(bx,by,bz,bw)\overrightarrow{b} = \overrightarrow{d} \times \overrightarrow{a} = (bx,by,bz,bw) b=d×a=(bx,by,bz,bw)
  6. 利用d→\overrightarrow{d}d取反求出c→=−d→\overrightarrow{c} = -\overrightarrow{d} c=−d
  7. (a→,b→,c→)(\overrightarrow{a},\overrightarrow{b},\overrightarrow{c})(a,b,c)即为视图矩阵中的(x→,y→,z→)(\overrightarrow{x},\overrightarrow{y},\overrightarrow{z})(x,y​,z)
