2788.按分隔符拆分字符串

链接：2788.按分隔符拆分字符串
难度：简单
标签：数组、字符串
简介：返回一个由拆分后的新字符串组成的字符串数组，不包括空字符串 。

题解 1 - typescript​

  • 编辑时间：2024-01-20
  • 执行用时：111ms
  • 内存消耗：59.37MB
  • 编程语言：typescript
  • 解法介绍：分割后平铺。
function splitWordsBySeparator(words: string[], separator: string): string[] {
    const sarr = separator.split('')
    return words
            .map(word => sarr.map(s => word.split(s)))
            .flat(3)
            .filter(Boolean)
};
